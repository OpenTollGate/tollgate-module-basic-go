# Protocol Specs — Moved

The TollGate protocol specification has moved to its canonical home:

**→ https://github.com/OpenTollGate/tollgate**

The spec files that previously lived in this directory (`TIP-01.md`,
`TIP-02.md`, `HTTP-01.md`, `HTTP-02.md`, `HTTP-03.md`, `NOSTR-01.md`,
`WIFI-01.md`) are now maintained in the repository above. File issues,
PRs, and spec-audit findings there.

The conceptual layering document (`transport-layer-distinction.md`) has
been folded into the upstream spec README's "Protocol Architecture"
section.
